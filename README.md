# 利用 npx create-react-app my-app 

- `npm start`

- `npm test`

- `npm run build`

- `npm run eject` 单项操作，拿回 webpack 等的配置

> 安装 antd 
`npm install antd --save`

> 安装 react-router
`npm install react-router-dom`
